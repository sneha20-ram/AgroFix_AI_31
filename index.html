Project Overview

Agrofix AI is a farmer-centric decision-support system designed to prevent crop losses by warning farmers about wrong actions, tracking outcomes, and showing cost savings. The system is optimized for **Tamil Nadu rural farming conditions**.

Unique Core Features

Do-Not-Do Risk Alerts

Warns farmers against risky actions like spraying before rainfall.

Decision Outcome Tracking

Farmer feedback is used to improve future recommendations.

Cost & Savings Analysis

Shows money saved by avoiding unnecessary inputs.

Small Farmer Focus

Designed for marginal farmers with low budgets.

Tamil Nadu Regional Support

Tamil Language Support

Interface and alerts available in Tamil for easy understanding.

Example: “இன்று பூச்சிக்கொல்லி தெளிக்க வேண்டாம்”

District-Based Weather

Weather alerts customized for Tamil Nadu districts.

Example: Rain alert for Thanjavur region.

Low-Cost Pesticide Locator

Shows nearby locations where pesticides are cheaper.

Today’s Risk Alert

Offline & Low-Signal Mode

Agrofix AI supports **offline usage for rural areas**. The system automatically syncs weather updates, alerts, and recommendations **whenever network signal is available**, instead of waiting for continuous internet connectivity.

This ensures uninterrupted access for farmers in remote villages.
